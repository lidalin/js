2. 基础base

基础部分请查看JavaScript语法及JS手册

对象是拥有属性和方法的数据。
JavaScript 中的所有事物都是对象：字符串String、数组Array、日期Date、逻辑Bool、算数Math、正则RegExp等等。

JS DOM
文档对象模型（Document Object Model）

  • 元素->节点
  • 属性
  • 样式
  • 事件

JS BOM
浏览器对象模型（Browser Object Model）
窗口Window、屏幕Screen、路径Location、历史记录History、浏览器Navigator、消息框PopupAlert、定时器Timing、Cookies
